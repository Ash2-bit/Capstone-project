# SAPULIDI - Sistem Pelaporan Bencana Alam Berbasis AI

## 📖 Tentang Proyek

SAPULIDI (Sistem Pelaporan dan Informasi Bencana Digital) adalah aplikasi pelaporan bencana alam berbasis web yang dikembangkan sebagai proyek akhir (Capstone Project). Platform ini memungkinkan masyarakat untuk melaporkan kejadian bencana secara cepat dan akurat serta membantu pihak terkait dalam melakukan pemantauan dan penanganan bencana.

Sistem ini memanfaatkan teknologi Artificial Intelligence (AI) untuk melakukan klasifikasi laporan secara otomatis berdasarkan informasi yang diberikan pengguna. Dengan demikian, proses identifikasi jenis bencana dapat dilakukan lebih cepat dan efisien.

## 🎯 Tujuan

* Mempermudah masyarakat dalam melaporkan kejadian bencana.
* Membantu proses klasifikasi laporan menggunakan AI.
* Menyediakan informasi bencana secara real-time.
* Mendukung pengambilan keputusan dalam penanganan bencana.

## ✨ Fitur Utama

* Pelaporan bencana berbasis lokasi.
* Klasifikasi jenis bencana menggunakan Artificial Intelligence.
* Dashboard monitoring laporan.
* Manajemen data laporan dan pengguna.
* Visualisasi lokasi kejadian pada peta digital.
* Sistem validasi dan verifikasi laporan.
* Statistik dan informasi bencana.

## 🚀 Instalasi

Clone repository:

```bash
git clone https://github.com/username/sapulidi.git
```

Masuk ke folder proyek:

```bash
cd sapulidi
```

Install dependencies:

```bash
npm install
```

Jalankan server development:

```bash
npm run dev
```

Buka browser:

```bash
http://localhost:3000
```
